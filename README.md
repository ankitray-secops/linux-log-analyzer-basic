# Linux Log Analyzer (Basic)

This is a beginner-level Python project that analyzes Linux authentication logs to detect failed login attempts.

## Why this project?
Log analysis is a core skill for SOC analysts. This script helps understand how brute-force and unauthorized login attempts appear in real Linux systems.

## What the script does
- Reads a Linux auth log file
- Detects failed login attempts
- Prints total failed attempts

## How to run
```bash
python3 analyzer.py
